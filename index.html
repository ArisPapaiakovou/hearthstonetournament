    Tournament Details

    The tournament will consist of two stages.

    The first stage, is a Round Robin that will be used to classify the participants for the second stage. The second stage is the Playoffs.

    Round Robin Stage

    In the Round Robin, every player will play one Match against every other player exactly once. A match consists of several game after which a match winner is declared.

    Prerequisites for a Match
    Player must prepare four Decks of four different classes. Decks have restrictions, please see Deck restrictions below for more details. Just before the start of the Match, each player Bans one and only one deck of this choosing of the opposing players four decks.
    Start of a Match
    Since each deck is represented by a Hearthstone character each player has now 3 heros they bring forward to do battle in this Match. Each player chooses their starting hero deck and a normal Hearthstone game begins. At the end of this first game there will be a loser (Hero died) and a winner (Hero is alive). The player with the dead hero can no longer use him again in this match and needs to bring forward the next Hero of his choosing to continue the battle. The winning player continues with his hero and a new game is started. The match continues like so until one player has killed all three of the opponents heros and is given the win of this match.
    End of match
    After the match is ended all heros are resurrected, including the banned hero, and a match can begin with another player.

    Playoffs stage

    Playoffs will use the ranking acquired from the round robin to seed the players. Say that four players participated in the round robin. First will play against the fourth, and second against the third in single elimination match format.

    Deck Restrictions

    Card Points Restrictions
    The deck must be build using a card point system. The Hearthstone disenchant value is used for the points. That is:
      40 points for a Common card
      100 points for a Rare card
      400 points for an Epic card
      1600 points for an Legendary card
      Basic cards have no value and can be used at will.
    Deck restrictions for future consideration
      Female characters cards only
      Human characters cards only
      Non-Human characters cards only
      Spell and Taunt cards only
      Common and Basic cards only

    First hearthstone tournament deck restrictions

    For our first tournament we will start with a simple 1500 Points Restriction.
